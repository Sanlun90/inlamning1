Inlämning 1 - Kurs: Databaser YH2024 - Skapad av: Sandra Bäckström
Detta repository innehåller alla mina filer för inlämningsuppgift 1 i kursen Databaser.

Beskrivning
Denna databas är skapad för att hantera en liten bokhandel. Den innehåller tabeller för:
Böcker
Kunder
Beställningar
Orderrader

Innehåll
inlamning1.sql - En SQL-fil med min exekverbara kod.
Inlämning1.drawio.png - En bild av ER-diagrammet som visar databasens struktur.

Instruktioner för användning
Öppna valfri MYSQL-klient.
Kör inlamning1.sql för att skapa och fylla databasen.
